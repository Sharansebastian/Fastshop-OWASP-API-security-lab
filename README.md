# FastShop-OWASP-API-security-lab
 Wallarm University – API Security Certified Associate. This is a free, hands-on certification course focused specifically on API security—one of the fastest-growing areas in cybersecurity. 
 https://wallarm.thinkific.com/products/courses/API-Security-Certified-Associate.
 Core learning approach:
  Practical, lab-based training (not just theory)
  Simulate real API attacks and learn how to stop them
  Built by real-world API security experts

The course has 3 chapters and 21 lessons including:

1. Setup & Environment:
Configure lab environment
Use tools like Postman
Set up Wallarm Security Edge

3. API Attack Techniques:
You’ll actually perform attacks like:
Broken Authentication
BOLA / IDOR (very important in APIs)
Rate limiting failures
Mass assignment
SSRF attacks
Business logic abuse

3. Defense & Detection:
Learn how attacks are detected
Understand mitigation strategies
Work with real security signals
